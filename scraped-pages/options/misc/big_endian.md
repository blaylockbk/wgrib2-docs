###

wgrib2: -big_endian -little_endian

## Introduction

The -big_endian and the
-little_endian
options changes the order that IEEE numbers are read and written. This
does not affect the order in which binary numbers are read/written.

## Usage

```

-big_endian
   or
-little_endian

```
