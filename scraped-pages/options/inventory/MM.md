
### wgrib2: -MM



### Introduction



The -MM option prints the month of the
reference time. The -MM 
and -YY options are pretty useless.

### Usage




```

-MM

```

### Example




```

$ wgrib2 png.grb2 -MM
1:4:MM=06

```
