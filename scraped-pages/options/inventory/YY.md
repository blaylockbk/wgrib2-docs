# wgrib2: -YY

## Introduction

The - option prints the year of the
reference time. The -MM
and -YY options are pretty useless.

## Usage

```
-YY
```

### Example

```
$ wgrib2 png.grb2 -YY
1:4:YY=2009
```

See also: [-t](./t.md), [-T](./T.md)
