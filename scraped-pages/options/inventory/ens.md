# wgrib2: -ens

## Introduction

The -ens option prints the ensemble information.

## Usage

```

-ens

```

### Example

```

$ wgrib2 -var-ens ens.grb
1:0:HGT:ENS=+1
2:45932:SPFH:ENS=+1
3:89724:SPFH:ENS=+1
4:144624:HGT:ENS=+1
5:198928:TMP:ENS=+1
6:219134:SPFH:ENS=+1
7.1:258504:UGRD:ENS=+1
7.2:258504:VGRD:ENS=+1
8:380298:SPFH:ENS=+1
9:441918:SPFH:ENS=+1

```
