# wgrib2: -lev

## Introduction

The -lev option prints the level of the
field.

## Usage

```

-lev

```

### Example

```

$ wgrib2 grib2.polar -lev
1.1:0:500 mb
1.2:0:850 mb

```

See also: [-s](./macros.md)

---

> Description: inv level (code table 4.5)
